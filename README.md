## 📌 О проекте

Это приложение, написанное на языке C, предназначено для работы на одноплатном компьютере **Luckfox Pico Ultra W** под управлением **Buildroot**. Основная задача программы — централизованный сбор и логирование данных в реальном времени с нескольких USB-устройств (до 24), подключенных к плате[reference:0].

Программа автоматически обнаруживает устройства в системе, инициализирует с ними связь по протоколу CDC (как с COM-портами) и сохраняет получаемые информационные дампы в специальные файлы на устройстве. Это решение идеально подходит для создания систем мониторинга или телеметрии.

## ✨ Ключевые особенности

*   **Поддержка до 24 устройств**: Программа сконфигурирована на работу с большим количеством подключённых одновременно USB-устройств[reference:1][reference:2].
*   **Автоматическое обнаружение**: При запуске сканирует и находит все устройства, представленные в системе как `ttyACM*`, и начинает работу с каждым из них[reference:3].
*   **Надёжное логирование**: Данные сохраняются в каталог `/userdata/dump_log/`, который сохраняется между перезагрузками устройства. Каждый USB-порт ведёт свой собственный лог-файл[reference:4].
*   **Потокобезопасность**: Для одновременной и стабильной работы с несколькими устройствами реализована многопоточность.
*   **Настраиваемый RAW-режим**: Для чтения данных с портов настроен корректный RAW-режим (с помощью `cfmakeraw`), обеспечивающий получение «чистых» данных без какой-либо обработки со стороны системы[reference:5].

## 🛠 Технологический стек
